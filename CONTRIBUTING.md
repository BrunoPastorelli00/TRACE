# Contributing to TRACE

Thank you for your interest in contributing to TRACE!

## Development Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```

## Development Guidelines

Please focus on:

* **Correctness over completeness** - Ensure what you build works correctly
* **Clarity over complexity** - Prefer simple, understandable solutions
* **Explicit limitations over implicit assumptions** - Document what the code does and doesn't do

## Code Style

* Use TypeScript with strict mode enabled
* Follow existing code patterns
* Add comments for complex logic
* Keep functions focused and small

## Testing

Before submitting a pull request:

1. Build the project: `npm run build`
2. Test the CLI tools with sample files
3. Verify that verification works correctly

## Submitting Changes

1. Create a feature branch
2. Make your changes
3. Test thoroughly
4. Submit a pull request with a clear description

## Areas for Contribution

* Core library improvements
* Additional media profile support
* Documentation improvements
* Bug fixes
* Performance optimizations

Thank you for contributing to TRACE!
